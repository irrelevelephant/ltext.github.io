How To Use It

Step 1: Have a Text File

The first thing you need is a text file you want to parameterize. This can be Html, JavaScript, Haskell, Bash, TeX, anything. In this example, we'll use a plantext file foo:

Step 2: Create a Parameter

Just like lambda calculus, we create arity via abstraction - the act of creating a parameter. In λtext it's very easy - all we need to do is, on the first line, declare our parameters inside delimiters. Then, we use those declared variables by referencing their names with the same delimiters. We are free to choose the syntax we wish - Jade-style delimiters with {{ }}, Html-friendly delimiters via comments <!-- -->, etc. Here's our new and improved foo: 

$$ x %%
Woo hoo! I'm a Text File!

$$ x %%

Step 4: Apply the Files/Functions with λtext

Lastly, to use our functions, we must apply one to another - the other counterpart to lambda calculus. To do this, invoke the ltext command from a terminal: 

$ ltext "foo bar"
