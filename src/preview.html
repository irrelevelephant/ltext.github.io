Reduce Duplication

Copy-paste is an essential keystroke for anyone using a computer, but when it's overused, it can cause serious maintainability headaches. With 位text, we can parameterize files themselves, and let substitution do the copy-paste'ing for us.

Built with Precision

With Lambda Calculus as it's underlying engine, 位text features both generality and precision - it's Hindley-Milner type system gives us the best of both worlds; polymorphism and correctness.

Open Source

位text is BSD Licenced, so you are free to do what you want with the project. It's written in the advanced programming language Haskell, with the code hosted on github.
