Partial Application

Just like lambda calculus, we can partially apply templates to each other. Take our apply template from before - that's a good example to work though. If we apply our foo file from before to apply, we should get a template from Text -> Text - the paramter to foo being unified with the x parameter in apply: 

$ ltext -t "apply foo"

apply foo :: Text -> Text

It turns out, we can actually render this partially applied template, but only if we explicitly supply new delimiters for the result: 

$ ltext "apply foo" --left "{{" --right "}}"

{{ x }}

asdf


Woo hoo! I'm a Text File!


{{ x }}

Sometimes newlines are added (due to the empty first line for files of type Text) - this is unavoidable for now.
