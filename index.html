λtext

General-Purpose Templating Utility

Reduce Duplication

Copy-paste is an essential keystroke for anyone using a computer, but when it's overused, it can cause serious maintainability headaches. With λtext, we can parameterize files themselves, and let substitution do the copy-paste'ing for us.

Built with Precision

With Lambda Calculus as it's underlying engine, λtext features both generality and precision - it's Hindley-Milner type system gives us the best of both worlds; polymorphism and correctness.
